## P1 单条质量

同一批 30 张真实产品图、相同业务目标与输出规格，在 Aivora 与同行各生成 30 条。文件名在评审前随机化，盲评 prompt 不含来源标识。

硬门槛：

1. Aivora VLM rubric 均分 ≥ 同行均分 × 0.95。
2. Aivora 黑屏、破图、时长错误、产品主体消失四类硬伤均为 0。
3. 提交数 = 成功 + 失败 + 取消，无静默丢失。

Rubric 固定维度：主体/产品一致性、几何与材质稳定、镜头意图完成度、动作与物理可信度、构图/光线、商业可用性。每维 0–5，并保留逐条理由。

## P2 模板稳定性

选 3 个高频模板，每个模板用同一组输入在两边各生成 20 条。报告每个维度的均值、标准差、P50/P95，并单独统计：

- 产品变形率；
- 品牌/包装文字伪影率；
- 手部/人物异常率（仅相关模板）；
- 跑偏率（未执行模板声明的镜头骨架）；
- 可直接交付率。

硬门槛：Aivora 总分标准差 ≤ 同行对应模板 × 1.2，且硬伤率不高于同行。

## 内部先行验证

同行对照到达前，Aivora 先完成：

1. 每个 ACTIVE 模板 20 个确定性 prompt 快照，确认无占位符、无不相关品类负面词、引用图顺序可审计。
2. 3 个代表模板 × 20 条 mock 全链路，验证分配、状态机、QA、入库和对账。
3. 真实 provider 仅在成本 Gate 与人工预算批准后运行；不得用 mock 结果宣称达到同行质量。
