# Aivora 宠物内容智能采集套件 · 60 秒产品讲解视频

> 投资人 Demo 配套视频。16:9 横屏，4 段 × 15 秒 = 60 秒。
> 画面由即梦/Seedance 生成（干净画面、**不在画面里渲染任何文字**），
> 中文字幕与 BGM 在 ffmpeg 拼接阶段烧录，保证中文显示正确、风格统一。

## 制作流程

```
npm run demo:gen:petkit       # 提交 4 段到 Seedance（需 BYTEPLUS_ARK_API_KEY）
npm run demo:check:petkit     # 轮询各段状态，全部完成后给出 segment URL
npm run demo:stitch:petkit    # 下载 → 归一化 → 烧中文字幕 → 拼接 → 加 BGM
```

- 状态记录：`tmp/pet-kit-walkthrough-video/submission.json`
- 成片输出：`public/generated/aivora-pet-content-kit-walkthrough-60s-16x9.mp4`
- 可选 BGM：设置环境变量 `PET_WALKTHROUGH_BGM=/绝对路径/bgm.mp3`（免版权）。
  未设置则成片为无配乐版本（静音），可后期再补。
- 字幕字体：默认 `/System/Library/Fonts/PingFang.ttc`，可用 `PET_CAPTION_FONT` 覆盖。

成片完成后，把最终 URL（或本地 `/generated/...mp4` 路径）写入
`src/lib/demo/pet-content-kit-demo-data.ts` 的 `PET_WALKTHROUGH_VIDEO_URL`，
hero 区会自动出现「可选 · 60 秒产品讲解」入口。

## 整体风格（Shared Style）

- 暖色、温馨、宠物友好、高级感；奶油白 / 暖米色 / 浅橙为主，深青绿点缀。
- 真实感家居场景，柔和自然光，浅景深，电影感但不浮夸。
- 宠物以猫 / 小型犬为主，表情可爱、动作自然。
- **画面内不出现任何文字、字幕、水印、logo、UI 文案、假品牌名。**
- 不出现人物正脸特写；如需人出现，用手部 / 背影 / 剪影。

## 分镜脚本（4 段）

### 段 1 · 0-15s · 真实瞬间被采集
- 中文字幕：**真实瞬间，自动被记录**
- 画面：温馨家中，一只可爱的猫在做日常的事（吃饭、窗边打盹、追玩具）；
  书架角落有一个小巧现代的家用摄像头静静地工作，暖色调，岁月静好的氛围。
- 传达：智能硬件在不打扰的情况下，自动捕捉宠物真实的可爱瞬间。

### 段 2 · 15-30s · AI 识别可爱与产品片段
- 中文字幕：**AI 自动挑出最可爱、最值得分享的片段**
- 画面：同一只猫的高光时刻——歪头、在柔软宠物垫上打滚；
  画面用柔和的光晕 / 聚焦框优雅地强调这些瞬间，暖色，有「被 AI 温柔注视」的感觉。
- 传达：AI 从长素材里自动筛选最可爱、最有传播力、最适合展示产品的片段。

### 段 3 · 30-45s · 自动生成可分享视频与宠物日记
- 中文字幕：**一键生成可爱视频和宠物日记**
- 画面：一部竖屏手机里播放着可爱的宠物短视频，旁边是一张温暖的宠物日记卡 / 心情卡；
  桌面暖色、有植物和毛毯，整体治愈。
- 传达：系统自动把高光瞬间剪成可分享的短视频，并生成宠物口吻的日记与心情卡。

### 段 4 · 45-60s · 分享裂变 + 品牌证据 + 社区
- 中文字幕：**分享带来增长，真实使用成为品牌证据**
- 画面：手机把宠物视频分享出去，多个宠物主人温暖的居家镜头快速串联；
  其中穿插宠物真实使用产品（趴在恒温垫上）的画面，结尾回到温馨的整体氛围。
- 传达：主人乐于分享形成低成本增长；真实使用场景同时成为品牌可信的产品证据，沉淀为社区。
