# ReelForge — 系统概览

## 项目定位

ReelForge 是一套面向商业客户的 **AI 短视频内容服务工作流系统**。

核心定位：**先做服务，再做账号矩阵**。第一阶段用于支撑"AI 短视频内容服务"的交付能力，而非面向 C 端的 SaaS 产品。

## 目标客户

- 需要 TikTok/YouTube Shorts/Instagram Reels 短视频内容的品牌方
- 中小企业的数字营销团队
- 需要批量高质量短视频内容的 MCN/代运营机构

## 核心能力

1. **标准化需求采集** — 将客户自由文本需求转化为结构化 Brief
2. **数据驱动策略** — 基于平台趋势和项目数据生成内容策略
3. **AI 脚本生成** — 针对平台特性生成优化的短视频脚本
4. **视频生成调度** — 统一调度多种 AI 视频生成服务
5. **自动化质检** — 规则引擎 + AI 联合质检
6. **人工审核流程** — 支持"北美前台 + 中国后台"的协作模式
7. **数据回流** — 交付后的表现数据沉淀，持续优化

## 商业模式

```
Phase 1: 服务交付 → 证明能力和效率
Phase 2: 流程标准化 → 降低交付成本
Phase 3: 账号矩阵 → 规模化运营
```

## 技术架构原则

- 所有关键模块结构化输入输出（JSON Schema）
- 能规则化的先规则化，不过度依赖 LLM
- 先半自动，后全自动
- 每个 Agent 职责单一明确
- 所有环节可追踪成本、质量和返工数据
