# RF-013 技术 token 豁免清单

本清单只允许技术标识在诊断、追踪、成本与文件格式语境中保持原文；所有客户可见的标题、按钮、导航与说明文案仍必须进入 i18n。

- `JOB ID`：登录页能力说明中的标识符字段名，以及任务追踪界面的字段标签。
- provider 名称：`BytePlus`、`Buddy`、`OpenAI` 仅在 provider 诊断、成本归因与调用链追踪中保留品牌原名。
- 协议、格式与工具 token：`API`、`URL`、`CSV`、`MP4`、`ffmpeg`、`VLM`、`QA`，仅在指代具体协议、文件格式或工程系统时保留。
- 状态与角色枚举：仅在 internal 诊断视图需要与数据库原始值逐字对照时保留；客户界面必须展示本地化标签。

以上豁免不得作为运营文案豁免；按钮、标题、导航、说明文字仍必须进入 i18n。
